# Telepathy

think you and twin are locked in? measure your telepathy with this two player game.

## Requirements

- Python 3.8+
- fastapi
- uvicorn

Install dependencies:

```bash
pip install fastapi uvicorn
```

2. Open the Game
   Open your browser and navigate to:
   👉 http://127.0.0.1:8000 (or http://localhost:8000)

3. Play with Two Players
   Open two separate browser windows (or tabs side-by-side) to http://127.0.0.1:8000.

Enter a unique player name in each window.

Start guessing clues in real-time to test your telepathy!
